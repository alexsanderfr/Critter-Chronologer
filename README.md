# Critter Chronologer

Web app that demonstrates database usage

### Prerequisites

```
Java
Spring
MySQL
```

### Installing

```
Clone this repo
Configure your MySQL server
Run with Spring
```

## Built With

* [Spring](https://spring.io/) - Spring framework
* [MySQL](https://dev.mysql.com/) - MySQL server
